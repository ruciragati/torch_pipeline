# Data Pipeline: CSV to PyTorch Tensor

## Overview
A modular Python pipeline that cleans raw CSV data and converts it into PyTorch Tensors.

## How to Run
Copy and paste these commands into your terminal to create the environment and install libraries:
1. Create venv: `python3 -m venv .venv`
2. Activate the environment: `source .venv/bin/activate`
2. Install: `pip install -r requirements.txt`
3. Run: `python main.py`
